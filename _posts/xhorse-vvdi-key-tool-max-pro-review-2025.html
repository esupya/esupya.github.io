You need to generate 10 new VVDI remote keys using the VVDI Max Pro function. After collecting these points, the server will automatically enable the ID48 96bit function approximately 2 hours later. All remotes must be brand new for this process.

The tool supports Corolla, Highlander, Camry, REIZ, VIOS, LEVIN, CROWN, RAV4, Hiraks, Lexus, and PRADO/LAND CRUISER models. Note that 8A-A9 type all key lost and 4A/8A-AA type add/all key lost scenarios are not currently supported.
