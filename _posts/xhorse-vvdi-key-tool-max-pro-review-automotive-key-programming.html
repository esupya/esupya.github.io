No, the device interface only supports English. If you require Russian language support, you'll need to purchase from a local dealer that offers localized versions.

How do I get the ID48 96bit transponder clone function?

You need to generate 10 new VVDI remote keys using the VVDI Max Pro. After collecting these points, the server will automatically enable the ID48 96bit function approximately 2 hours later. Note that all remotes must be brand new.

What Toyota models are currently supported?

The device supports Corolla, Highlander, Camry, REIZ, VIOS, LEVIN, CROWN, RAV4, Hiraks, Lexus, PRADO, and LAND CRUISER models. Note that 8A-A9 type all key lost and 4A/8A-AA type add/all key lost are not currently supported.

Is the MINI OBD Tool included or required?

Can I use this with Xhorse key cutting machines?
